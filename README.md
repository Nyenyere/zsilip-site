# Zsilip public landing page

This repository contains a copyrisk-clean public landing page for the Zsilip edge-network research prototype.

The page is intentionally conservative. It does not publish internal logs, credentials, screenshots, phone numbers, SIM data, MAC addresses, private endpoint details, or research-pack excerpts.

Positioning: Zsilip is presented as a research and validation prototype for multi-uplink edge networking, client observability, proof-first status reporting, and controlled demonstration workflows.

The static site uses only original HTML, CSS, and SVG assets. There are no external images, icon packs, fonts, CDN dependencies, vendor logos, or copied competitor material.
